# Professor Pranab Panday — Academic Portfolio

A world-class academic portfolio site built with **React**, **Tailwind CSS**, **Framer Motion**, and **React Router**. Style: academic + product-grade (serif + sans typography, deep navy/charcoal, emerald accent, generous whitespace, 12-column grid).

## Features

- **Home**: Hero, impact stats (count-up), research themes (global filter), featured work, career timeline, awards & partners, CTA strip
- **About**: Summary, expertise, education, global engagements
- **Research**: Overview + research clusters (filter drives Publications)
- **Publications**: Tabs (Books / Articles / Chapters / Reports / Conference), Fuse.js search, year + tag filters, Copy APA / BibTeX / DOI
- **Teaching & Supervision**: Courses by level, supervision list, examiner roles, prospective students
- **Projects & Grants**: Grant cards with filters
- **Consultancy**, **Leadership & Service**, **Awards**, **Media & Talks**, **Contact** (form + Calendly)
- Sticky nav with **Download CV**, dark/light mode toggle
- Global research-theme filter (Home + Research + Publications)
- Accessibility: skip-to-content, keyboard-friendly, WCAG-minded contrast

## Run

```bash
npm install
npm run dev
```

Build: `npm run build` → output in `dist/`.

## Content

- Edit **`src/data/content.js`** for profile, stats, publications, grants, awards, etc.
- Put your **CV PDF** at **`public/cv.pdf`** so “Download CV” works.

## Stack

- React 19, Vite 7, Tailwind CSS 3, Framer Motion, React Router, Fuse.js
